Cómo implementarias lo anterior en este código. El div card contiene una imagen de placeholder y al hacer hover se muestra la imagen real de data-hover. El div.card es la caja que contiene los elementos del portafolio, pero en este caso la imagen es pixel art, no todos serán pixel art, solo algunos, ¿Cómo lo puedo implementar de manera individual?


#CSS
/* Custom Card Styles */
/* Card */
.grid > .card {
  background-color: white;
  border-radius: 0.5rem;
  padding: 1rem;
  transition: transform 1.2s ease;
}

.grid > .card:hover {
  transform: translateY(-4px);
}

/* Image Container */
.responsive-image {
  position: relative;
  width: 100%;
  aspect-ratio: 3 / 2; /* Mantener proporción fija */
  background-color: #f0f0f0;

  overflow: hidden;
  margin-bottom: 1rem;
  cursor: pointer;
}

/* Placeholder (SVG) */
.responsive-image img {
  width: 100%;
  height: auto;
  transition: opacity 0.4s ease;
}

/* Imagen real (fondo) */
.card .responsive-image::before {
  content: '';
  position: absolute;
  top: 0; left: 0;
  width: 100%;
  height: 100%;
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
  opacity: 0;
  z-index: 1;
  transition: opacity 0.4s ease;
  border-radius: 0.35rem;
}

.card:hover .responsive-image img {
  opacity: 0;
}

.card:hover .responsive-image::before {
  opacity: 1;
}
